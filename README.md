# Eccentric Essays

A professional essay writing service platform built with Next.js 16, React 19, and Tailwind CSS.

## Getting Started

This project uses **pnpm** as the package manager. If you don't have pnpm installed, install it first:

```bash
npm install -g pnpm
```

### Installation

Install dependencies using pnpm:

```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

Build the application for production:

```bash
pnpm build
```

### Start Production Server

Start the production server:

```bash
pnpm start
```

## Features

- Responsive design with mobile-first approach
- User authentication system
- Dashboard with inbox, tasks, calendar, and analytics
- Order management system
- Real-time messaging inbox
- Supabase backend integration

## Tech Stack

- **Framework:** Next.js 16
- **UI Library:** React 19
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI
- **Database:** Supabase
- **Package Manager:** pnpm
